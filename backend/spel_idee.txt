Het Spel
Iedereen scant bij aankomst hun QR-code. De server weet nu wie meedoet.
Doel: als eerste 10 codes oplossen.

Per ronde
Je bezoekt 4 stations (gsm's/tablets). Elk station vertelt je welk station je daarna moet bezoeken — zo verdeelt de server de load en blijft iedereen bewegen.
Op elk station krijg je één van twee dingen:

Een tip — informatie die je nodig hebt om de code op te lossen
Een kenmerk — een actie die je moet uitvoeren zodat je partner je herkent, bv. "loop achterwaarts", "doe de macarena", "leg je hand op je hoofd"

Je krijgt 3 tips en 1 kenmerk per ronde. Het kenmerk kan vroeg of laat komen, dat beslist de server op basis van wanneer er een partner beschikbaar is.

Partner vinden
Jij voert de actie uit, je partner zoekt iemand die die actie doet. Zodra jullie elkaar bevestigen, delen jullie jullie tips. Jullie hebben samen 6 tips.

Code oplossen
Pas als je alle 3 tips hebt én je partner gevonden hebt, kun je de code invullen. Elke ronde is de code anders van type:

+ Som van de getallen
- Morse-code decoderen
- Weegschaal in balans
+ Squid Game brug
- Mijnenveld

Je weet tijdens het ophalen van je tips nog niet welk speltype het wordt. Schrijf alles op papier want de app toont tips niet opnieuw.
Fout? Je moet naar een ander station gaan om opnieuw te proberen. Je tips blijven geldig, maar je verliest tijd. Je kunt ook kiezen om je 3 tips opnieuw op te halen — maar dat kost nog meer tijd.
Juist? Volgende ronde, nieuwe partner, nieuw speltype.

De eerste 2 spelers die alle 10 rondes voltooien winnen.
